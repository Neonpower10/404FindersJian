FR-006: Openingsperiodes instellen
 
User story:
Als campingbeheerder wil ik mijn eigen openingsperiodes kunnen instellen zodat mijn campingwebsite altijd de juiste beschikbaarheid laat zien.
 
Definition of Ready (DoR):
1. Het is duidelijk hoe periodes worden opgeslagen (bijv. start- en einddatum).
2. Er is bekend hoe de periodes worden weergegeven op de website.
 
Definition of Done (DoD):
1. Openingsperiodes kunnen worden ingevoerd en opgeslagen.
2. De ingevoerde periodes worden correct weergegeven op de campingwebsite.
 
FR-007: Beheeromgeving
 
User story:
Als campingbeheerder wil ik via één centrale beheeromgeving mijn campinggegevens kunnen beheren zodat ik alles overzichtelijk kan aanpassen.
 
Definition of Ready (DoR):
1. Het is duidelijk welke gegevens in de beheeromgeving moeten staan (naam, faciliteiten, prijzen, etc.).
2. Er is bekend hoe de beheeromgeving toegankelijk wordt (login, rolbeheer).
 
Definition of Done (DoD):
 
1. Alle campinggegevens zijn vanuit één omgeving aanpasbaar.
2. Wijzigingen worden direct doorgevoerd op de campingwebsite.
 
FR-008: Camping inrichten op eigen stijl
 
User story:
Als campingbeheerder wil ik de kleuren, afbeeldingen en teksten van mijn website kunnen aanpassen zodat de site past bij mijn eigen stijl.
 
Definition of Ready (DoR):
 
1. Het is duidelijk welke onderdelen van de stijl aanpasbaar zijn (kleur, logo, foto’s, tekst).
2. Er is bekend hoe dit technisch wordt opgeslagen en toegepast.
 
Definition of Done (DoD):
 
1. Campingbeheerders kunnen hun stijl aanpassen in de beheeromgeving.
2. De wijzigingen worden correct weergegeven op de website.
 
FR-009: Werkt voor alle soorten campings
 
User story:
Als campingbedrijf wil ik dat het systeem geschikt is voor campings van elke grootte zodat het platform voor iedereen bruikbaar is.
 
Definition of Ready (DoR):
 
1. Het is duidelijk welke soorten campings worden ondersteund (groot/klein).
2. Er zijn geen harde limieten die gebruik verhinderen.
 
Definition of Done (DoD):
 
1. Het systeem functioneert correct ongeacht het aantal plekken of faciliteiten.
2. Tests bevestigen dat zowel kleine als grote campings het systeem kunnen gebruiken.
 
FR-012: Boeken
 
User story:
Als kampeerder wil ik online een kampeerplek kunnen reserveren zodat ik zeker weet dat er plaats is.
 
Definition of Ready (DoR):
 
1. Het is duidelijk welke gegevens nodig zijn voor een boeking (naam, datum, plek).
2. Er is bekend hoe een boeking bevestigd wordt (bijv. e-mailbevestiging).
 
Definition of Done (DoD):
 
1. Boeken is mogelijk via de website.
2. Boekingen worden correct opgeslagen en zichtbaar in de beheeromgeving.
